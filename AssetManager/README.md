# 💼 AssetManager v1.0

A real-time enterprise management application designed to track corporate hardware inventory, analyze reserve allocations, and manage internal network directory user access.

## 🛠️ Tech Stack
* **Frontend:** Angular (Standalone Components, RxJS streams, Tailwind CSS)
* **Backend:** .NET Core Web API (C#)
* **Database:** SQL Server via Entity Framework Core (Code-First)

## ✨ Core Features
* **Live Analytics Dashboard:** Unified view tracking total corporate assets, monetary value, and unassigned reserves using parallel RxJS execution streams.
* **Reactive Directory Management:** Dynamic form-to-table system with auto-refreshing CRUD workflows for tracking system users.
* **Toast Notification Engine:** Custom visual validation events indicating data updates across asynchronous component state changes.
